# Link para inscrição

Clique [aqui](https://docs.google.com/forms/d/1_TNHPAkoEejj-My7uS-jkkk4q18hvBCnh7jLk_gIYAA/edit) para se inscrever no evento.

*	04/04/2022 – Encerramento das inscrições

*	19/05/2022 – Primeiro dia de evento
